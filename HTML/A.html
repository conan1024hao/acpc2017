Problem A: Clock

Problem

xx君はお気に入りの時計を持っている。
ある日時計の長針が取れてしまい、どこかへなくしてしまった。しかし、xx君はその時計を使い続けたいため短針だけで時刻を読み取りたいと考えている。
短針の情報θが与えられた時の時刻(時h、分m)を出力せよ。

  • 時計はいわゆるアナログ時計で、1から12の数字が12を一番上として時計回りに昇順に並んでいるものである。
  • 短針の情報は12を0°として整数で時計回りを正の向きで与えられる。

Input

入力は以下の形式で与えられる。

  θ

Constraints

  • 角度は度数法(degree)で表される。
  • 0 ≤ θ ≤ 359

Output

短針が与えられた角度をとる時の時刻を
h m
の形で一行に出力せよ。

時刻に午前午後の区別はないため0 ≤ h ≤ 11とする。

Sample Input 1

0

Sample Output 1

0 0

Sample Input 2

45

Sample Output 2

1 30

Sample Input 3

100

Sample Output 3

3 20
