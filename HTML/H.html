Problem H: Making Pairs

Problem

招待制の店があって、最初は店主だけがいる。
あるタイミングで、店主またはそれまでに招待された人のうち、誰か一人が新規客を招待する。
誰かに招待された人は必ず参加する。
すでに参加している人を招待することはない。
異なる人が同時に招待されることはない。
この店では二人組で行う出し物があるが、各人はコミュ障なので、
自分が招待した人もしくは自分を招待した人とだけペアを組むことができる。
新たに人が招待され参加するたびに、その時点で最大何ペア作ることができるのかを求めたい。

Input

入力は以下の形式で与えられる。

N
p1
p2
...
pN
一行目に招待される人数の合計 N が与えられる。
続く N 行に pi 番目に参加した人が i 番目の人を招待したという情報が与えられる。 ( 1 ≤ i ≤ N )
ただし、店主は 0 番目に参加した人として扱うこととする。

Constraints

入力は以下の条件を満たす。

  • 1 ≤ N ≤ 5000
  • 0 ≤ pi < i ( 1 ≤ i ≤ N )
  • N, pi は整数

Output

出力は N 行からなる。
i 行目に i 番目の人が参加した時点で最大何ペア作ることができるのかを出力する。 ( 1 ≤ i ≤ N )
出力の最後には改行を入れること。

Sample Input 1

3
0
0
2

Sample Output 1

1
1
2

Sample Input 2

5
0
1
2
3
4

Sample Output 2

1
1
2
2
3
